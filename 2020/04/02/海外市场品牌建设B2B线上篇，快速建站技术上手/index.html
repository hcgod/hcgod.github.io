海外市场品牌建设B2B线上篇，快速建站技术上手

发表于 2020-04-02 更新于 2020-04-03 分类于 海外市场品牌建设B2B 文章阅读次数：

前面的文章聊了太多虚的，这里讲讲实际的建站技术 来点实际的。
随着网络技术的发展呢，现在建站是越来越容易了。很多soho都自己建站，自己优化。
正好最近因为疫情的原因，自己公司项目和帮朋友做了一些项目 有些感悟，这里我顺便写写。 其实外贸行业里面，做外贸品牌建设的，大多数都不是技术出生。
卖东西不错，但是要建站就是一头雾水了。 找人做，动不动就是几万的开支就出来了，很多刚开始的soho又觉得不划算。 这里我简单说说怎么做吧。

为了有更好的切入点，这里我们还是以企业官网为切入点来说。 其他的什么电子商城啊，后面再说吧，恰好后面公司要打算开始这一块的业务。

对于官网，大部分不涉及支付啊，库存管理啊之类的，更多的都是CMS的内容管理。 及时发布公司新闻啊，最多做做产品的询盘啊什么的。

基于这个需求，其实我最推荐的是采用wordpress程序建站，四个字，成熟稳定！
那我们要如何做出一个尽可能SEO友好的官网呢？建网站四部曲：域名-> 服务器 -> 部署 -> 上线。

域名

关于域名类型的影响

两类 从最初的gTLD .com, .cn, .net等； 到现在的new gTLD: .site, .shop, .xyz 等等
关于域名类型对SEO的影响，总说纷纭。 刚开始流行new gTLD的时候，纷纷传说这个新的类型对搜索引擎不友好。 后来有传说，new gTLD因为比较新没有黑历史，所以容易获得较高排名。
其实这些都没有得到证实，也没有数据支撑。 域名类型，选择的时候还是尽可能贴近品牌形象吧。

域名

如何选择域名注册商？

都差不多，国内阿里巴巴，国外godaddy 都可以任意选，这个不影响，只要不得跑路。 因为后面最主要的域名管理服务会交给hosting那边，所以注册商随便选。只是煮一下，如果是阿里云那边注册得话最好实名制一下，不然修改域名服务器得时候要很久才生效。 有实名制分分钟神效。

域名服务器DNS

一般你注册域名之后，你的域名服务器默认是在域名注册商那里，你在你域名注册商的管理后台就能看到。 域名服务器，可以简单的理解为 谁来提供域名管理服务。 域名注册服务是由注册商来提供，比如阿里云，腾讯云，狗daddy等。 那域名管理服务谁来提供呢？ 一般默认就好， 但是这里我们会建议你将你的域名服务器地址修改为你的hosting服务商（下面讲服务器会提到）。 阿里云是在 域名，域名管理，修改DNS里面。 注意，不是DNS解析哈。 解析这个动作是由你的域名管理服务提供方来做的，如果你的域名DNS是在阿里云，那你就在阿里云解析，如果你的域名DNS指向了其他管理服务商就在其他管理服务商后台去解析（我们这里一般推荐你解析到hosting服务商那边，方便做CDN加速和SSL）

网站服务器

选弹性服务器VPS还是网页托管hosting服务？

基于官网，这里我推荐hosting。 因为一个弹性VPS服务器，相当于是一个云端得电脑，里面什么都没有，你还需要自己去部署网站环境，很麻烦 而且贵！ 而且VPS提供的算力，你的官网一般是用不完你的，所以完全没必要。 如果你的企业OA或者其他企业级系统对运算要求更高的，那VPS便是很好的选择。

bluehost，inomtionhosting 还是siteground?

在hosting服务里面，现在很成熟，比较著名的由bluehost, inmotionhosting, siteground, 这三个我都用过。 针对官网，我推荐siteground. 三家价格都差不多，主要是siteground是有专门针对wordpress程序优化的。 用起来非常便捷流畅，从SSL到CDN加速，轻松几部就能搞定。 要买就买不限站点数量那个版本，扩展性能都不错。

什么是SSL？

就是给你的网站链接颁发一个安全认证，浏览器地址栏开头出有一把锁。 有SSL安全认证的就是绿色的锁，可以经由https访问。 如果没有认证过，就是一个红色的叉叉，提示网站不安全。 你想嘛，人家都看到你的域名不安全了，谁还敢给你发询盘？ 所以，这个很重要。

什么是CDN 加速？

就是让你的网站在全球不同地方得节点生成缓存镜像，这样离那个地方近得客户访问速度就飞起~ 节点越多越安逸！ 当然，免费的一般都3~5个，siteground后台集成了这个CDN加速得，是cloudflare提供得服务。 在后台一键加速起飞。

部署网站

注册完成hosting之后，就可以通过hosting后台得一键安装包安装想要wordpress。 安装完后就可以通过网站后台开始管理你的网站了。 后台一般是 域名/wp-admin

美化

安装好后，去后台找个主题安装上，推荐去themeforest买个付费得主题，然后上传到后台就行。然后填充自己的内容。

功能

其他功能都是由plugin插件实现的，想要什么功能就去插件市场找找。 基本上一个企业官网需要的功能，或者你不需要的功能他都有。 只是记住，不用的插件就清理掉。
